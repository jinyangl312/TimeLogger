# MytTime
这是一个可以记录你有效学习工作时间的小软件，由python实现，简陋但有点用

# 已实现功能
- 点击开始计时
- 点击结束计时
- 结束自动保存记录
- 历史记录加载
- 每天的记录都会保存【以json格式保存】
